wellrestd

This is a JavaScript implementation of the wellrestd tool defined in http://restd.net/tools.html. I cannot recommend using this version of the tool on very large files as it may crash or lock your browser.

As input, paste the content of an entire restd file in this first box.

Then, click one of these buttons. They correspond to the wellrestd arguments.
